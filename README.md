## Description
This is my go at attempting to create a snake game using WPF and XAML, with the help of https://wpf-tutorial.com/. 


## Main Features
- Snake moves
- Create food
- Settings menu
- Death screen
- Eat food
- Score
- Collision detection
- Variable speed
- Difficutlies
